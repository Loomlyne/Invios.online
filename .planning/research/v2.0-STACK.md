## Core Stack (Unchanged)

| Database | Supabase Postgres + RLS | New tables via migrations |
| Auth | Supabase Auth | Single-user per account |

---

## Additions by Phase

### Phase 13–19 (No new vendors required)

- **Slug routing:** Next.js dynamic segments only
- **Time tracking:** Postgres + server actions
- **Automation:** JSONB rules + cron (same as Phase 5)
- **Forecast:** Pure TypeScript on existing analytics helpers
- **Proposals:** Extend quotations table + public page UI
- **Webhooks:** `fetch()` from server actions with HMAC — no Zapier SDK required for v2.0

### Phase 20 — AI Co-pilot

| Component | Recommendation | Rationale |
|-----------|----------------|-----------|
| LLM access | **Vercel AI Gateway** (`ai` package, model string) | Already on Vercel; no extra Stripe-like signup |
| Structured output | `generateObject` + Zod | Matches invoice line item types |
| Streaming | `streamText` for review explanations | Optional UX polish |
| Rate limiting | Per-user daily cap in `profiles` or KV | Cost control for side project |

**Do not add:** OpenAI SDK directly unless user requests — use AI SDK per project Vercel guidance.
